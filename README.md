# QHAT: Quantum Hamiltonian Analysis Toolkit

The QHAT ("Q-hat") software package is developed by Los Alamos National Laboratory to support
research into quantum computing.  It is written in Python and provides a variety of tools related
to Hamiltonian simulation on quantum computers.  The current pieces are

- **`hamiltonian_generator`:** A tool that takes the description of a molecule and generates a
  Hamiltonian.
- **`qre`:** Resrouce estimation for quantum circuits involving Hamiltonians.
- **`common`:** Components used by other parts of QHAT.  Users will typically not interact
  directly with files in this directory.

Documentation is provided as a README file in each directory.

If you need support or have a feature request, the best way to reach out to the team is to open an
issue in the GitHub repository.
